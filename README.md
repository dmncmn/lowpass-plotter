# lowpass-plotter

Test program for visualisation of working lowpass FIR filter.
N - filter order
fc - cutoff frequency
f - generated frequency
tau - experimental value


Run:
```
python -m src.main
```

Run tests:
```
python -m unittest -v tests.tests
```
